# Setup Checklist

1. Install Node.js LTS (>=18.0.0) and npm/pnpm.
2. Install Expo CLI: `npm install -g expo-cli` (or use `npx expo`).
3. Clone repo and run package install: `npm install` or `pnpm install`.
4. Copy `.env.example` to `.env` and provide Supabase values.
5. Run typecheck: `npm run typecheck`.
6. Run lint: `npm run lint`.
7. Run tests: `npm run test`.
8. Start development server: `npm run start`.
9. Run `npm run android` (requires Android Studio emulator) or `npm run ios` (macOS + Xcode).
10. Validate initial screens render in your emulator/device.
